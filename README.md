# Airbnb Market Intelligence Analysis

## Overview

This project was completed as part of a Data Engineering Internship Technical Assessment.

The project analyzes the New York City Airbnb dataset obtained from the Inside Airbnb platform and generates business insights through data cleaning, transformation, and exploratory data analysis.

## Technologies Used

- Python
- Pandas
- Matplotlib
- Visual Studio Code
